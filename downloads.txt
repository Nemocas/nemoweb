<ul>

</ul> 

<p>To obtain Nemo, follow these steps:</p>
<ul> 
<li>Install <a href="http://julialang.org/downloads/#release-candidates-v040-rc3">Julia-0.4</a> or later.</li>
<li>julia> Pkg.clone("https://github.com/wbhart/Nemo.jl")</li>
<li>julia> Pkg.build("Nemo")</li>
<li>julia> using Nemo</li>
</ul> 

<p>To run the Nemo test suite:</p>
<ul>
<li>julia> using Nemo</li>
<li>julia> Nemo.Test.test_all()</li>
</ul>
